Три консольные игры на языке C

Учебный проект — три классические консольные игры, реализованные на языке C с поддержкой русского языка в Windows (Visual Studio).

Состав проекта

| Файл             | Игра            | Описание                                            |
| ---------------- | --------------- | --------------------------------------------------- |
| `guess_number.c` | Угадай число    | Угадай загаданное число от 1 до 100                 |
| `hangman.c`      | Виселица        | Угадай слово по буквам, пока не закончились попытки |
| `tictactoe.c`    | Крестики-нолики | Игра для двух игроков на поле 3×3                   |

Требования

* Компилятор: MSVC (Visual Studio 2019/2022)
* ОС: Windows
* Стандарт: C99 или выше

Запуск в Visual Studio

1. Открыть Visual Studio.
2. Создать новый проект «Консольное приложение (C)».
3. Скопировать содержимое нужного `.c` файла в `main.c`.
4. Убедиться, что файл сохранён в кодировке UTF-8.
5. Нажать F5 для запуска.

Каждая игра представляет собой отдельный `.c` файл и отдельный проект.

Описание игр

Угадай число (`guess_number.c`)

Программа загадывает случайное число от 1 до 100. Игрок вводит варианты, программа подсказывает: «слишком мало» или «слишком много». В конце выводится количество потраченных попыток.

Возможности:

* Случайное число при каждом запуске
* Подсчёт количества попыток
* Защита от некорректного ввода

Виселица (`hangman.c`)

Угадай загаданное слово из тематики программирования. Даётся 6 попыток. При каждой ошибке виселица дополняется новым элементом.

Словарь:

* программа
* компилятор
* массив
* функция
* указатель
* структура
* алгоритм
* переменная

Возможности:

* Виселица из символов консоли
* Отображение уже введённых букв
* Работа с русскими словами

Крестики-нолики (`tictactoe.c`)

Классическая игра для двух игроков за одним компьютером. Поле 3×3, клетки выбираются по номеру от 1 до 9.

Пример поля:

| 1  | 2   | 3  |
| -- | --- | -- |
| 4  | 5   | 6  |
| -- | --- | -- |
| 7  | 8   | 9  |

Возможности:

* Проверка победы по строкам, столбцам и диагоналям
* Определение ничьей
* Защита от выбора занятой клетки
